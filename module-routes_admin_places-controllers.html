Methods

(async, static) importPlaces(req, res)

Importe le planning des inspecteurs pour un département avec un fichier CSV ou XLSX. Le planning est une liste de places contenant les informations dans l'ordre suivant: date, heure, matricule de l'inspecteur, le nom de l'inspecteur, le nom du centre et le département. La fonction métier appelée est importPlacesFromFile
